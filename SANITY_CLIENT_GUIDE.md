# Sanity CMS Setup - Client Guide

## What is Sanity?
Sanity is your new content management system. It's much easier and more secure than the previous GitHub-based system. You'll have a beautiful, user-friendly interface to manage your church content.

## Access Your Content Manager

**Studio URL**: http://localhost:3000/studio (when running locally)
**Production URL**: https://christrevolutionministries.org/studio (after deployment)

## Getting Started

### 1. First Time Login
- Open the Studio URL
- You'll be prompted to sign in with Google or GitHub
- Use your existing Google account or create one

### 2. Content Types Available

#### **Ministries**
- Add new ministries
- Edit descriptions, leaders, meeting times
- Upload ministry photos
- Set contact information

#### **Sermons** 
- Add sermon titles, preacher names
- Set sermon dates and series
- Add YouTube/audio links
- Write descriptions and scripture references
- Tag sermons for easy searching

#### **Events**
- Create upcoming events
- Set dates, times, and locations
- Add event descriptions and images
- Mark events as featured on homepage
- Set registration requirements

### 3. How to Add Content

1. **Click the "+" button** or select a content type
2. **Fill in the required fields** (marked with *)
3. **Upload images** by dragging and dropping
4. **Click "Publish"** to make content live on the website

### 4. Managing Images
- Drag and drop images directly into image fields
- Sanity automatically optimizes images for web
- Add alt text for accessibility

### 5. Publishing Content
- **Draft**: Content saved but not live
- **Published**: Content appears on website immediately
- Changes appear on the website within 1-2 minutes

## Key Benefits Over Previous System

✅ **No Technical Knowledge Needed** - Just point and click  
✅ **Secure** - No sharing of credentials or tokens  
✅ **Real-time** - Changes appear almost instantly  
✅ **Mobile Friendly** - Edit content from your phone  
✅ **Image Optimization** - Automatic image resizing and optimization  
✅ **Version History** - See all changes and revert if needed  
✅ **User Roles** - Multiple people can have access with different permissions  

## Need Help?
- Content appears on the website automatically after publishing
- If you make a mistake, you can always edit or unpublish content
- Contact your web developer if you need additional content types or features
